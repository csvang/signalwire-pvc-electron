# SignalWire PVC Electron POC

Integrating SignalWire's Programmable Video Conference into Electron

## Setup

- Using Node v16.15.1+
- Install Node modules
`npm i`
- Start Electron app
`npm start`

## Build App

- Tested on Mac.  Creating *.app - `npm run make`
